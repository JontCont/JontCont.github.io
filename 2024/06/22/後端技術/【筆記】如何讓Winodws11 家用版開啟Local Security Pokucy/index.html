前言

原本我需要把 檔案總管 左側icon 移除，發現 Windows 11 家用版沒有 Local Group Policy，需要使用指令列的方式來開啟。

一、打開記事本

複製下面的指令列，貼到記事本，存檔為 LocalGroupPolicy.bat。

二、以系統管理員身分執行

右鍵 LocalGroupPolicy.bat 選擇 以系統管理員身分執行，等待安裝完成。

三、開啟 Local Security Policy
